G-Drive Show

Mache dein Google Drive zur Diashow

G-Drive Show ist eine Android-App, die deine Fotos und Bilder aus Google Drive automatisch als Diashow abspielt – einfach verbinden, auswählen und genießen. Keine eigenen Server, keine Datenspeicherung.

Google Drive API OAuth 2.0 drive.readonly Android Keine eigenen Server
